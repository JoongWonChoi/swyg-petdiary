## ✅백엔드 개발 과정
### 📌사용 기술 스택
* Java 11 jdk
* Spring framework
* SpringBoot version 2.6.6
* JPA Hibernate
* dependencies
  + Spring Security
  + Spring Data JPA
  + Spring Web
  + Lombok
* 개발 과정에서는 H2 Database 사용
* 후에 MySql로 교체 예정
